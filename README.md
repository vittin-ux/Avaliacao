#Sistema de Gerenciamento de Alunos -- Projeto em Python

Este projeto contém um sistema simples em Python que utiliza
**estruturas de repetição**, **condições**, **listas** e
**dicionários**.\
O objetivo é manipular alunos e suas notas através de um menu
interativo.

------------------------------------------------------------------------

### Cadastrar Aluno

Armazena o nome de um aluno dentro de uma lista, usando um dicionário.

### Adicionar Nota

Procura um aluno pelo nome e adiciona uma nota na sua lista interna.

### Listar Alunos

Mostra todos os alunos cadastrados.

### Exibir Alunos com Notas

Exibe cada aluno, suas notas e a média calculada.

### Sair

Encerra o sistema.

------------------------------------------------------------------------

## Exemplo de execução

    Menu:
    1 - Cadastrar aluno
    2 - Adicionar nota
    3 - Listar alunos
    4 - Exibir alunos com notas
    5 - Sair

Saída esperada:

    João - Notas: [8.5, 7.0] | Média: 7.75
    Maria - Notas: [9.0] | Média: 9.00
